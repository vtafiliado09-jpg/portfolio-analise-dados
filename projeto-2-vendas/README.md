# 🛒 Dashboard de Análise de Vendas

## 📊 Sobre o projeto
Dashboard desenvolvido no Power BI para análise de vendas de e-commerce.

## 🎯 Objetivo
Analisar desempenho de vendas por região, categoria e tempo.

## 📈 Análises realizadas
- Evolução das vendas ao longo do tempo
- Vendas por categoria
- Lucro por região

## 💡 Insight
A categoria Technology apresenta maior volume de vendas.
